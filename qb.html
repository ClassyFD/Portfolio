No 1 About Project

QuiltBack Gives Back

QuiltBack is a website that was built with ReactJS. Its main purpose is to allow users to communicate by creating blogs, and host events to raise money through auctioning quilts. This website was designed by a UI/UX team, and was made scalable and responsive by a group of 3 handsome developers.

No 2 The Process

Starting Out

At first, we started with 4 developers: 2 frontend, 2 backend. After a couple of days, the other frontend developer stopped showing up. I had to take control of the frontend and finish all tasks that were assigned to 2 frontend developers in a span of 2 weeks.


Half-Way Point

I was constantly working with the backend devs to ensure that all endpoints were setup correctly and usable. I was consistantly working to develop the frontend, as well as juggling any other side-tasks that I had to finish. By the end of the first week, We had successfully set up most of the user functionality and now were moving on to the admin side.

Finishing Up

In the end, we successfully reached our mvp mark and were ready to present our project. We are currently working with the client to find any features we would need to implement and any bugs that we need to squash.
